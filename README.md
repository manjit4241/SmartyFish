# 🐟 SmartyFish
A modern, visually appealing React Native (Expo) application designed to interface with an ESP32-based Smart IoT Fish Feeder. Control feeding parameters, view history logs, and monitor your aquarium with ease.

## 🚀 Features
- **Dashboard**: Real-time monitoring of feeding schedules and status.
- **Hardware Configuration**: Seamlessly configure and connect to your ESP32 hardware.
- **Feeding Logs**: Keep track of the feeding history, including times and amounts.
- **Aesthetic UI**: Smooth animations and a beautiful, aquarium-themed design built with React Native.

## 🛠️ Built With
- **[React Native]((https://reactnative.dev/))** & **[Expo](https://expo.dev/)**
- **[Lucide React Native](https://lucide.dev/icons/)** for beautiful icons
- **[Moti](https://moti.fyi/)** & **[React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/)** for smooth, 60fps animations
- **[Axios](https://axios-http.com/)** for robust HTTP communication with the ESP32

## 📦 Getting Started
1. Clone the repository: `git clone https://github.com/manjit4241/SmartyFish.git`
2. Install dependencies: `npm install`
3. Run the application: `npx expo start`

## 📡 Hardware Integration (ESP32)
The application expects an ESP32 to be hosting a simple API to configure and trigger feedings. To learn more about setting up the hardware, refer to the IoT implementation details (Documentation coming soon).
